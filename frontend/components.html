🌐 국제화 & 접근성

다국어 지원과 WCAG AA 준수.

언어 선택

language 선택 시 data-i18n 속성이 있는 모든 요소가 번역됩니다.

접근성 (WCAG 2.1 AA)

  • Skip Link: 페이지 최상단 "메인 콘텐츠로 건너뛰기" 링크 (Tab 키로 접근 가능)
  • Focus Ring: 모든 상호작용 요소는 focusvisible 스타일 적용 (brand-500 색상)
  • Aria Labels: 아이콘 전용 버튼에는 aria-label 적용
  • Color Contrast: 모든 텍스트는 4.5:1 이상의 명도 대비 확보
  • Reduced Motion: prefers-reduced-motion 설정 시 모든 애니메이션 비활성화

애니메이션 감소 모드

활성화 시 모든 transition/animation 토큰이 무시되고 변화가 즉시 적용됩니다.

참고: 모든 토큰은 frontend/css/tokens.css에서 정의됩니다. 페이지 CSS에서 하드코딩된 색상/값 대신 항상 토큰을 사용하세요.
