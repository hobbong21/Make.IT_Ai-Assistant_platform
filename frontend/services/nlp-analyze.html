자연어 분석

텍스트·파일·URL을 분석하여 감정 분포, 키워드 빈도, 요약을 자동 추출합니다. 여러 건 비교 분석과 결과 내보내기를 지원합니다.

1. 템플릿 (선택)

자주 쓰는 분석 시나리오를 클릭하면 텍스트 영역에 즉시 채워집니다.

2. 입력

입력한 URL의 본문을 추출하여 분석합니다.

3. 고급 설정

4. 결과

5. 최근 이력

최근 10건이 이 브라우저(localStorage)에 저장됩니다. 항목을 클릭하면 결과가 복원됩니다.
